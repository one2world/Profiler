# Profiler-2025 程序员开发工具集合平台

## 📌 项目概述

Profiler-2025 是一个基于 **Python + PySide6** 的程序员日常开发辅助平台，采用模块化设计，提供多种开发工具的统一入口。平台以工具集合为核心，每个工具均为独立模块，可独立运行、互不影响，具备现代美观的 GUI 及良好的模块扩展性。

### 核心特点

- 🔌 模块化设计：每个工具为独立模块，可单独开发、测试和运行
- 🎨 现代化界面：基于 PySide6 构建，提供美观的 GUI
- 🔄 可扩展性：便于添加新的工具模块
- 🎯 独立运行：各模块数据、状态、UI 互不干扰

## 🛠️ 功能模块

### 1. 主程序框架

- **统一入口**：提供所有工具的分类导航和启动功能
- **模块管理**：动态加载和管理工具模块
- **界面布局**：
  - 顶部工具栏：全局设置、主题切换、帮助等
  - 左侧分类列表：按功能分类展示工具
  - 中间工具列表：显示当前分类下的所有工具
  - 状态栏：显示操作状态和提示信息

### 2. 内存分析器模块

#### 2.1 数据分析功能

- **堆栈信息解析**
  - 支持多种堆栈格式
  - 解析地址、模块、函数名、源码位置等信息
  - 构建完整调用路径

- **内存分配分析**
  - 记录大小、地址、对象类型等信息
  - 支持按帧统计和分析
  - 提供多维度统计数据

#### 2.2 可视化功能

- **火焰图**
  - 按堆栈路径合并分配数据
  - 支持大小比例显示
  - 交互式查看和选择

- **调用树**
  - 分层显示堆栈结构
  - 显示分配大小和次数
  - 支持搜索和过滤

- **时间线**
  - 按帧显示内存分配趋势
  - 支持点击选择具体帧
  - 显示帧内存分配详情

#### 2.3 交互功能

- **视图切换**：火焰图/调用树双视图自由切换
- **数据联动**：视图间数据同步和联动更新
- **选择反馈**：状态栏显示选中项详细信息

## 🚀 开发指南

### 环境要求

- Python 3.8+
- PySide6
- pyqtgraph

### 安装依赖

```bash
pip install PySide6 pyqtgraph
```

### 运行项目

1. 启动主程序：
```bash
python main.py
```

2. 在主界面中选择并启动具体工具模块

### 开发新模块

1. 创建模块目录结构
2. 实现 BaseModule 接口
3. 按照模块命名规范组织代码
4. 在主程序中注册新模块

## 📝 开发规范

### 模块开发规范

1. 模块必须实现 BaseModule 接口
2. 保持模块独立性，避免互相依赖
3. 提供完整的模块文档
4. 实现必要的测试用例

### 代码风格

1. 遵循 PEP 8 规范
2. 使用类型注解
3. 编写详细的文档字符串
4. 保持代码整洁和可维护性

### UI 设计规范

1. 遵循现代化界面设计原则
2. 保持界面简洁清晰
3. 提供必要的交互反馈
4. 支持键盘操作
